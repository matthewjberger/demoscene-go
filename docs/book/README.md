# indigo Documentation

Comprehensive documentation for the [indigo](https://github.com/matthewjberger/indigo) game engine, built with [mdBook](https://rust-lang.github.io/mdBook/).

## Building

Install mdBook:

```bash
cargo install mdbook
```

Build and serve locally:

```bash
mdbook serve --open
```

Or from the repository root:

```bash
just book-serve
```
